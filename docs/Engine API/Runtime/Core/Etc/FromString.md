# FromString

FromString is a static struct that handles converting a string into different value types

## Functions

``` cpp
static int32 FromString::Int(String& Input)
```
Converts `Input` to int and returns it

---

``` cpp
static double FromString::Double(String& Input)
```
Converts `Input` to double and returns it

---

``` cpp
static float FromString::Float(String& Input)
```
Converts `Input` to float and returns it

---

``` cpp
static bool FromString::Bool(String& Input)
```
Converts `Input` to boolean and returns it
