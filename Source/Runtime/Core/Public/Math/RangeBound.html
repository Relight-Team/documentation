Core::Math::RangeBound

Core::Math::RangeBounds handles the RangeBound Class

RangeBound contains a value for the Range class to use for upper and lower bounds

There are 3 types of Bounds:

  • Inclusive - A bound that includes the borders values (i.e 1-5 would be [1, 2, 3, 4, 5])
  • Exclusive - A bound that excludes the borders values (i.e 1-5 would be [2, 3, 4])
  • Open - A bound that continues on forever (i.e an open bound of 1 would be [1, 2, 3, 4, 5, 6, 7, 8...])
