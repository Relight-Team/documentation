Array

Array handles the Array Class

Array is a custom Relight class that can store multiple values of the same type, unlike C's version, this one can be treated as both fixed size and dynamic size

Class Array (Template)

Array(Template Repeat, int Size)

A constructor that repeats the same value X amount of times

Array(int Size)

A constructor that set's the default size

int Num()

Returns the size of the array, starts at 1

int Size()

Returns the size of the array, starts at 0

void Print()

A debug feature that prints each value in the array, it should print something like this:

0 | 1 | 2 | 3 |

bool Contains(Template Input)

Returns true if Input was found in the array

bool Find(T Input, int& Value)

If Input exist in array, store the index of the first instance into Value, return's true if successful

bool FindLast(T Input, int& Value)

If Input exist in array, store the index of the last instance into Value, return's true if successful

bool GetTypeSize()

Returns the size of the array type

bool IsValidIndex(int Index)

If the index is valid, return True, otherwise, return false

T Last()

Return the last item

T Last(int a)

Return the item backwards by a, (for example, if a = 1, it would return the second to last item)

T Top()

Return the first item

T Top(int a)

Return the item upwards by a, (for example, if a = 1, it would return the second item)

void Add(T Input)

Adds the Input to the array

void Append(Array& Input, int Size)

Will get each value of the Input up to the size amount and adds them to the array

void SetNum(int NewSize)

Change the size of the array, if the new size is shorter than the current size, then it will erase each desired value backwards

void AddUnique(T Input)

Adds the Input to the array only if the Input doesn't exist in the Array

void Insert(T Input, int Index)

Adds the Input to the array at an index

void RemoveAt(int Index)

Remove the item from Index

bool RemoveSingle(T Input)

Remove the first instance of Input, returns true if successful

void Remove(T Input)

Remove all instances of Input

void Empty()

Remove all items and set size to 0 in array

Operators

[] operator (ArrayName[int index])

Returns the item from the index

= Operator (Array1 = Array2)

Converts the first array into the second array

+= Operator (Array1 += Array2)

Appends the first array and adds all values from the second array
