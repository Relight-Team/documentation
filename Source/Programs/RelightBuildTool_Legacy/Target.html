Relight Engine Documentation

Target files

Target files are files that end in **.Target** and tells RBT how to build the project.

Commands

Name = String

The name of the target, this will also be the name for the final executable

Target = String

Will both place and configure the file depending on it's target

  • Game = The entire compiled code located in the project's own "bin" directory, mainly for games
  • Full = Like Game but stores it in the "Engine" bin directory, mainly for engine itself
  • Client = Like Full but with server code excluded
  • Server = Like Full but with Client code excluded
  • Editor = The compiled code created for expanding the Relight Editor
  • Software = Code that is built on top of Relight Engine

ExtraDependencies = [String Array]

An array of dependencies that the target will compile together, these usually will access the .Build file from within them
