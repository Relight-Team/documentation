RBT and Configs

The RBT software and it's config

How to Build target files

In order to build a target file, you must type this command in your terminal

python3 [Path/to/RBT]/main.py [Target] [Platform]

Argument: Target

Type: String

Description: the path to the target file

Argument: Platform

Type: String

Description: The platform that the target will compile to

  • Win64 = Windows x64
  • Unix = Unix systems (linux, freeBSD, etc)

Config

The config file will be generated once RBT run's for the first time. It contains important information for compiling targets, all global configs are located in **GlobalCfg.py**

Targets

RBT_Ver = [int, int, int]

contains the current RBT's version

Engine_Directory = string

The Directory of engine without a "/" at the end. RBT will attempt to auto-generate this

Compiler = string

The compiler that RBT will use, 'g++' is default

ShowDebug = boolean

If true, it will show the command being used when compiling, useful for debugging, the default value is false

How to include a third party project into Relight engine

1. Create a folder in the /ThirdParty directory

/ThirdParty/Example/

2. Create 2 folders in your third party project director, Implement and Include

/ThirdParty/Example/Implement - Your implementation files goes here

/ThirdParty/Example/Include - Your include files goes here

3. Put your .a file in 'Implement' and your .h files in 'Include'

4. Add both implement and include into your build file (go to build instructions for more info)

5. Include your include file in your code
